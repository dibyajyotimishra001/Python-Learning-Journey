# Student Grade Calculator

A Python program to calculate a student's total percentage and assign a grade based on marks obtained in Physics, Chemistry, and Mathematics.

## Description
This script prompts the user to input their marks for three subjects. It includes input validation to ensure marks are between 0 and 100, and uses a `try-except` block to handle non-numeric inputs gracefully. The program then calculates the total percentage, checks if the student meets the minimum passing criteria (at least 33 in each subject and an overall 40%), and assigns a specific grade accordingly.
