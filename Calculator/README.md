# Simple Arithmetic Calculator

A robust Python-based calculator designed to perform essential arithmetic operations while ensuring a crash-free user experience through rigorous error handling.

## Key Features
- **Core Operations:** Supports Addition, Subtraction, Multiplication, and Division.
- **Input Sanitization:** Uses `try-except` blocks to handle non-numeric inputs gracefully.
- **Zero Division Protection:** Explicitly checks and prevents division by zero errors to ensure stability.
- **Modern Output:** Utilizes Python's f-strings for clear and readable results.

## How to Use
1. Ensure you have Python 3.x installed on your system.
2. Run the script from your terminal:
   ```bash
   python calculator.py
