# **Quick-Quiz**

A simple and interactive command-line quiz application written in Python to test your general knowledge.

## Features

* **Interactive Questions:** Prompts users with a series of general knowledge questions.
* **Case-Insensitive Checking:** Accepts answers regardless of whether they are typed in upper or lower case.
* **Automated Scoring:** Keeps track of your correct answers and displays the final score at the end of the quiz.

## Prerequisites

* Python 3.x installed on your system.

## How to Run

1.  Create a new file named `quick_quiz.py`.
2.  Copy and paste the provided Python script into the file and save it.
3.  Open your terminal or command prompt.
4.  Navigate to the directory where your file is saved.
5.  Run the program using the following command:
    
    ```bash
    python quick_quiz.py
    ```

## Usage Instructions

Upon running the script, you will be prompted to answer a series of questions one by one. Ensure you type your answers in a single word as instructed and press **Enter**. Your final score out of the total questions will be revealed after all questions have been answered.
