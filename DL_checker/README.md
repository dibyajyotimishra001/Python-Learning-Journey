# Driving License Eligibility Checker

A clean and robust Python script to check driving license eligibility based on user age according to standard rules.

## Features
- Input Validation Uses try-except block to prevent crashes from invalid inputs.
- Clean Code Written following Python best practices (PEP 8) without unnecessary nested conditions.
- Self-Documenting Code is easy to read and understand.

## How to Run
1. Make sure you have Python installed on your system.
2. Clone this repository or download the `dl_checker.py` file.
3. Open your terminal or command prompt.
4. Run the script using the following command
   ```bash
   python dl_checker.py
