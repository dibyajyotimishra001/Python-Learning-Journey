# Random Joke Generator and Speaker

A fun Python script that generates a random joke and speaks it out loud.

## Description
This program utilizes the `pyjokes` library to fetch a random programming or tech-related joke. It then uses the `pyttsx3` library, an offline text-to-speech conversion tool, to read the joke aloud to the user. This is just a prototype even you can create a news reader and other projects

## Prerequisites
To run this script, you will need Python installed on your system along with the following external libraries:
- `pyjokes`
- `pyttsx3`

You can install the required libraries using pip:
```bash
pip install pyjokes pyttsx3
