# **To-Do-List**

A simple, interactive command-line application written in Python to help you manage your daily tasks effectively. It uses a infinite loop and ask you what to do? Features are: 1. Add a Task, 2. View Task, 3. Delete a Task, 4. Exit. When you click on exit it ends the loop.

## Features

* **Add a Task:** Easily append new tasks to your list.
* **View Tasks:** Display all current tasks with their corresponding index numbers. If the list is empty, the program will notify you.
* **Delete a Task:** Remove a specific task from your list by selecting its assigned number.
* **Exit:** Safely terminate the application.

## Prerequisites

* Python 3.x installed on your system.

## How to Run

1.  Create a new file named `todo.py` (or any other name you prefer).
2.  Copy and paste the provided Python script into the file and save it.
3.  Open your terminal or command prompt.
4.  Navigate to the directory where your file is saved.
5.  Run the program using the following command:
    
    ```bash
    python todo.py
    ```
